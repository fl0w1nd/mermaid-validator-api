Mermaid Validator

编辑器 API 文档

Mermaid 语法验证 API，使用 mermaid.js 在服务端本地解析校验，无需依赖外部渲染服务。所有端点均位于 /api 路径下。

GET /api/health

健康检查接口，用于确认服务是否正常运行。

响应
{
  "ok": true,
  "service": "mermaid-validator-api"
}

POST /api/validate

验证单个 Mermaid 图表语法。支持传入原始代码或 ```mermaid``` 代码块，会自动提取。

字段类型说明
codestringMermaid 代码 必填
idstring自定义标识，原样返回 可选
请求
{
  "id": "test-1",
  "code": "graph TD; A-->B"
}
响应
{
  "id": "test-1",
  "valid": true,
  "error": null
}

POST /api/validate/batch

批量验证多个 Mermaid 图表，单次最多 200 条。所有条目并发验证。

字段类型说明
itemsarray验证条目数组，每项包含 code 和可选 id 必填
请求
{
  "items": [
    { "id": "1", "code": "graph TD; A-->B" },
    { "id": "2", "code": "invalid code" }
  ]
}
响应
{
  "ok": true,
  "count": 2,
  "invalidCount": 1,
  "results": [
    { "id": "1", "valid": true, "error": null },
    { "id": "2", "valid": false, "error": "..." }
  ]
}
